<h1 align="center">
Fitness and Well-being
</h1>
<p align="center">
MongoDB, Expressjs, React/Redux, Nodejs
</p>

Fitness and Well-being is an easy-to-use fitness tracking web application. Its main goal is to assist users in their wellness journey, providing tools and features to help them achieve their fitness goals.

Your journey to wellness starts here!

Developed by Atindra Dubey

## Prerequisites

- MonggoDB
- Node JS
- NPM

## Fitness and Well-being Features

- Personal Account.
- Diet Profile.
- Goal Settings.
- Meal Planner.
- Water Intake Log.
- Workout Database.
- Nutrition Checker.
- BMR Calculator.

## Getting Started
Clone the repo
```shell
git clone https://github.com/Atindra28/Fitness-and-Well-Being
cd Fitness and Well-being
npm install
```

Install Frontend Dependencies
```shell
cd frontend
npm install
```

Setting the .env file
```shell
cp .env.example .env
nano .env
```
add your MonggoDB URI

## Run The App
Go Inside Root Folder, then run
```shell
npm run dev
```

## PORT
```shell
Frontend: 3000
Backend/API: 9000
```
